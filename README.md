# Linux Server Monitoring Tool

A lightweight monitoring tool for Linux servers that tracks system resources such as CPU, memory, disk usage, and system logs.

This tool helps system administrators monitor server health and detect potential issues early.

## Features

- Monitor CPU usage
- Monitor memory (RAM) usage
- Monitor disk usage
- Monitor system logs
- Alert when resources exceed thresholds
- Simple command-line interface

## Technologies

- Python
- Linux system utilities
- Bash scripting
- System monitoring libraries

## Example Metrics

The tool monitors:

- CPU utilization
- Memory usage
- Disk usage
- System log errors
- Server uptime
